Introduccin:

Este proyecto busca analizar y predecir la esperanza de vida utilizando datos recopilados de 193 pases entre los aos 2000 y 2015, provenientes de la Organizacin Mundial de la Salud (OMS) y la Organizacin de las Naciones Unidas (ONU).

El conjunto de datos contiene 22 columnas y 2938 registros, con 20 variables predictoras divididas en cuatro categoras principales:

Inmunizacin: Cobertura de vacunas como la hepatitis B y el sarampin. Mortalidad: Tasas de mortalidad infantil, materna y de enfermedades especficas. Economa: Producto Interno Bruto (PIB) per cpita y gasto en salud. Factores sociales: Nivel de desarrollo del pas y acceso a servicios bsicos.

Dado que los datos contienen valores nulos y valores atpicos, se ha realizado un proceso de limpieza en el que se imputaron los valores faltantes utilizando la mediana con la funcin SimpleImputer, debido a la presencia de outliers que podran sesgar la media. Adems, se eliminaron pases con datos insuficientes como Vanuatu, Tonga, Togo y Cabo Verde, asegurando as una base de datos ms robusta para el anlisis.

El objetivo principal del proyecto es clasificar la esperanza de vida en dos categoras:

  • Menor a 65 aos
  • Mayor o igual a 65 aos

Para ello, se aplicarn distintos modelos de clasificacin, como Anlisis Discriminante Lineal (LDA), regresin logstica y rboles de decisin con poda, asegurando una correcta seleccin de caractersticas mediante tcnicas estadsticas. Se evaluarn los modelos a travs de mtricas de desempeo en los datos de prueba y se compararn los resultados para determinar el modelo ms preciso en la prediccin de la esperanza de vida.

Con este estudio, se espera comprender mejor los determinantes clave de la longevidad y generar herramientas predictivas que puedan apoyar la toma de decisiones en salud pblica y planificacin econmica.

import pandas as pd
from sklearn.model_selection import train_test_split

# Cargar el dataset
data = pd.read_csv("Life_Expectancy_Dat_A2.2.csv")

# Verificar las columnas disponibles
print(data.columns)

data = data.dropna(subset=['Life expectancy '])

# Convertir la columna 'Life expectancy ' en una variable categrica

data['Life_Expectancy_Category'] = pd.cut(

import statsmodels.api as sm

# Preparar las variables independientes y dependientes
X_train = train_data.drop(columns=['Life expectancy ', 'Life_Expectancy_Category', 'Country', 'Status'])  # Eliminamos columnas no numricas
y_train = train_data['Life_Expectancy_Category'].map({'Baja': 0, 'Alta': 1})  # Convertimos a binario

# Asegurarnos de que todas las columnas en X_train sean numricas
X_train = X_train.apply(pd.to_numeric, errors='coerce')

# Eliminar columnas que no se pudieron convertir a numricas (si las hay)
X_train = X_train.dropna(axis=1)

# Aadir una constante a las variables independientes
X_train = sm.add_constant(X_train)

# Ajustar el modelo de regresin logstica
logit_model = sm.GLM(y_train, X_train, family=sm.families.Binomial())
result = logit_model.fit()

# Imprimir el resumen del modelo
print(result.summary())

# Seleccionar las dos variables ms relevantes basadas en los valores p
# Supongamos que las dos variables ms relevantes son 'Adult Mortality' y 'Income composition of resources'
p_values = result.pvalues.drop('const')  # Excluimos la constante
selected_variables = p_values.nsmallest(2).index.tolist()  # Seleccionamos las 2 variables con los p-values ms pequeos

print("Variables seleccionadas:", selected_variables)

# Filtrar los datos de entrenamiento y prueba con las variables seleccionadas
X_train_selected = train_data[selected_variables]
X_test_selected = test_data[selected_variables]

from sklearn.tree import DecisionTreeClassifier, plot_tree
from sklearn.model_selection import LeaveOneOut, cross_val_score
import numpy as np

# Seleccionar el valor ptimo de alpha usando LOOCV
alphas = np.arange(0.01, 0.1, 0.01)
cv_scores = []

for alpha in alphas:
    tree = DecisionTreeClassifier(ccp_alpha=alpha)
    scores = cross_val_score(tree, X_train_selected, y_train, cv=LeaveOneOut())
    cv_scores.append(np.mean(scores))

optimal_alpha = alphas[np.argmax(cv_scores)]

# Ajustar el rbol de decisin con el alpha ptimo
optimal_tree = DecisionTreeClassifier(ccp_alpha=optimal_alpha)
optimal_tree.fit(X_train_selected, y_train)

# Visualizar el rbol resultante
plt.figure(figsize=(20,10))
plot_tree(optimal_tree, filled=True, feature_names=selected_variables, class_names=['Baja', 'Alta'])
plt.show()
